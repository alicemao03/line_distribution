Introduction

The goal of this project is to visualize the line distribution of the 13-member K-Pop boy group SEVENTEEN (SVT).

Matching lyrics to members

The first step was to find who sings which parts. colorcodedlyrics.com is community-driven repository of Kpop songs and their lyrics and which member is singing it. I utilized the python library Beautiful Soup to scrape each of the SVT songs. However, because the site relies on diverse contributors, the data lacked a standardized schema. This presented a significant data-cleaning challenge. I had to develop a normalization logic to reconcile different style guides, standardize member aliases, and filter out non-lyrical metadata to ensure a consistent dataset across all tracks.

Finding the timestamps

The next step was finding the timestamp for when each line occurs. Some of the official music videos have close captions (CC), so with downsub.com I was able to get the precise start and ending timestamps for each line. For the songs that did not have available CCs, I use the open-sourced LRCLIB which provided the start times for each line. However, since it did not provide end times, I just used when the next line appears as the end of current line. This isn't as accurate and a further implementation of this project would be to update to correct end times.

Merging it all together

The hardest part was merging the color coded lyrics and the timestamp lyrics together. Since the format of both lyrics were different it was often the case that adlibs or shortened apprebiations would occur in one set of lyrics and not the other. I used a two-pointer approach to merge the lines together and used FuzzyWuzzy to determine matching lines. The results would the be writen to a JSON file in the /json/songs file of this repo.

Rendering it on html

Finally, using D3.js, I was able to visualize the line distributions for each song for each member. The home page is a display of each song. Clicking on the song can also give you a more indept comparsion between members.
